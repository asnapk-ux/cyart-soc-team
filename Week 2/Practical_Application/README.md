# Practical Application
This folder contains the practical SOC activities completed during Week 2.

## Completed Tasks:
## 1. Alert Management Practice
   - Alert classification using Google Sheets
   - Priority assignment using CVSS scoring
   - Mapping alerts to MITRE ATT&CK techniques
   - Visualization of alert priorities using Wazuh dashboards
   - Incident ticket creation and escalation documentation

## 2. Response Documentation
   - Incident response template based on SANS format
   - Investigation steps and activity logs
   - Phishing investigation checklist
   - Post-incident review and lessons learned

## 3. Alert Triage Practice
   - Analysis of security alerts in Wazuh
   - Priority assessment and status assignment
   - Threat intelligence validation using VirusTotal

All detailed reports and evidence for the completed tasks are available
in the PDFs and Screenshots folders.
